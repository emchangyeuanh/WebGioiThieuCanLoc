# Thư mục Music

Đây là thư mục lưu trữ các file nhạc MP3 cho trang web.

## File cần có:
- **dan-ca.mp3** - File nhạc dân ca Ví, Giặm chính

## Hướng dẫn:
1. Thay thế file placeholder bằng file MP3 thực tế của bạn
2. Đặt tên file chính xác là `dan-ca.mp3`
3. Định dạng: MP3
4. Kích thước: Tối ưu dưới 10MB để tải nhanh

## Cách thêm file:
1. Copy file MP3 vào thư mục này
2. Hoặc upload từ máy tính vào thư mục này
3. Trang web sẽ tự động phát khi nhấn "Nghe Ngay"
